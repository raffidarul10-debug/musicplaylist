🎵 Music Playlist

A simple and modern web-based music player built with HTML.

✨ Features

- Play and pause music
- Next & previous track
- Progress bar
- Responsive design
- Simple and clean UI

🛠️ Built With

- HTML5
- CSS3
- JavaScript

🚀 How to Run

1. Download or clone this repository.
2. Open "index.html" in your browser.
